# Ser - 是 - shì

## 🧩 Decomposição
* 是: [[wiki/radicais/日 - Sol]] (日 — sol/dia) no topo + 疋 (pǐ — pé/base) abaixo. "O sol que nasce todos os dias É uma verdade absoluta."

## 🧠 Conexões Neurais
* **Negação**: 不是 (bú shì — não é/não ser) com [[wiki/vocab/Lote 1/Não - 不 - bù]]
* **Confirmação**: 是的 (shì de — sim, é verdade)
* **Padrão**: Sujeito + 是 + Substantivo (estrutura básica de identificação)
* **Estrutura de pergunta**: 是...吗？com [[wiki/vocab/Lote 2/Partícula Interrogativa - 吗 - ma]]
* **Conceito**: [[wiki/conceitos/Estrutura SVO]] — 是 como cópula

## 💡 Mnemônica
"Assim como o **sol** (日) nasce todos os dias acima dos nossos pés, isso **É** (是) uma verdade absoluta."

## 💬 Contexto (HSK1)
*(Padrão: Sujeito + 是 + Substantivo)*
1. **他是老师吗？**
   * *Tā shì lǎoshī ma?*
   * Ele é professor?
2. **我不是中国人。**
   * *Wǒ bú shì zhōngguó rén.*
   * Eu não sou chinês.
3. **这是什么？**
   * *Zhè shì shénme?*
   * O que é isto?
