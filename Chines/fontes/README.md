# fontes/ — Fontes Brutas

Esta pasta contém **material de origem imutável**. O LLM lê daqui, mas **nunca escreve ou modifica** nada aqui.

## Como usar

1. Salve aqui artigos, PDFs convertidos, anotações de aula, transcrições de vídeo, etc.
2. Diga ao LLM: *"ingest fontes/nome-do-arquivo.md"*
3. O LLM processará e atualizará o wiki automaticamente.

## Subpastas sugeridas

- `fontes/artigos/` — artigos e leituras em geral
- `fontes/aulas/` — notas de aula, transcrições
- `fontes/listas/` — listas de vocabulário HSK oficiais
- `fontes/assets/` — imagens baixadas de fontes externas

## Convenção de nomenclatura

`YYYY-MM-DD - Título Descritivo.md`

Ex: `2026-05-01 - Lista Oficial HSK1 Completa.md`
